# Usage & Instalation

Instalation
---
git clone https://github.com/PR0PH3CY0x1/xss_punisher.git

cd xss_punisher

Setup
---

pip3 install -r requirments.txt

python scanner.py -h

Usage
---

Try flag '-h' to see all the help message.
